# ML — Repeated Question Finder 🎯🌸

Upload up to 10 PDF / Word (.docx) question papers. The app finds questions that repeat 2+ times, shows how many times each one appears per file, and lets you export the results. 💡📚✨

## 🚀 Features
- Drag & drop or click to upload up to 10 `.pdf` / `.docx` files
- Detects repeated questions (2+ times) across all uploaded papers
- Shows per-file repeat counts (which paper, how many times)
- Search questions by keyword
- Filter: only show questions repeated across multiple files
- Filter: only show questions repeated 3+ times
- Export results as PDF or Word (.doc)

## 🛠️ Tech
- HTML, CSS, JavaScript (no backend — runs fully in the browser)

## 📖 How to use
1. Open `index.html` (or the GitHub Pages link once deployed)
2. Drop in your question papers (PDF or Word)
3. Click **Find repeated questions**
4. Use search / filters to narrow results
5. Export as PDF or Word

## 🎯 Goal
Help students quickly spot the most frequently repeated questions across previous years' question papers.
